Lists
    - Ordered sequences that can hold a variety of object types
    - They use [] brackets and commas to separate objects in the list
        - [1,2,3,4,5]
    - Lists support indexing and slicing
    - Can be nested and also have a variety of useful methods

    - sorted() - function, which returns a new list that is sorted
    - .sort() - method that sorts the list in place
