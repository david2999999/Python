Dictionaries
    - unordered mappings for storing objects using key-value pairs
    - This key-value pair allows users to quickly grab objects without needing
      to know the index location
        - Uses curly braces and colons to signify the key and their associated values
            - {'key1':'value1', 'key2':'value2'}
    - Objects are retrieved by key name
    - Unordered and can not be sorted
